### 7、堆

- 堆分为两种：*最大堆*和*最小堆*，两者的差别在于节点的排序方式 
  - 在最大堆中，父节点的值比每一个子节点的值都要大。
  - 在最小堆中，父节点的值比每一个子节点的值都要小。这 就是所谓的“堆属性”，并且这个属性对堆中的每一个节点都成立。
